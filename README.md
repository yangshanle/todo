# 📦 Washi Portfolio

一个简洁优雅的个人作品集网页，以和纸（Washi）美学为灵感，融合现代设计风格。

## 🌐 在线地址

[https://yangshanle.github.io/todo/](https://yangshanle.github.io/todo/)

## ✨ 功能

- **🏠 首页** — 展示头像、简介、精选作品、最新文章、技能标签
- **💼 Works 作品页** — 卡片网格展示，支持搜索、标签筛选、详情弹窗
- **📝 Articles 文章页** — 文章列表，支持搜索、分类筛选、全文弹窗
- **👤 About 关于页** — 个人介绍、统计图表、时间线、语录、技能、联系方式
- **🖼 Gallery 相册页** — 图片网格，支持全屏查看器翻页浏览
- **🔒 编辑模式** — 密码保护，支持添加/编辑/删除内容、拖拽排序
- **🌙 暗色模式** — 一键切换明暗主题
- **⌨️ 键盘快捷键** — E=编辑、1-5=页面切换、Esc=关闭弹窗
- **🖱 灯箱 & 查看器** — 点击图片放大，Gallery 专用全屏翻页查看器

## 🛠 技术栈

纯前端 SPA，无框架依赖：

- **HTML** — 语义化结构
- **CSS** — 自定义属性主题、Grid 布局、渐变装饰、毛玻璃效果、响应式设计
- **JavaScript** — MVC 模式、localStorage 持久化、IntersectionObserver 动画、拖拽排序

## 🚀 使用

所有数据存储在浏览器 localStorage 中。点击导航栏 `✎` 按钮，输入密码 `1` 进入编辑模式即可管理内容。
