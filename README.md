# 📦 Washi Portfolio

一个简洁优雅的个人作品集网页，以和纸（Washi）美学为灵感，融合现代设计风格。

## 🌐 在线地址

[https://yangshanle.github.io/todo/](https://yangshanle.github.io/todo/)

## ✨ 功能

- **🏠 首页** — 展示头像、简介、精选作品、最新文章、技能标签，带视差滚动背景与装饰公式
- **💼 Works 作品页** — 卡片网格（桌面 3 列，平板 2 列，手机 1 列），支持搜索、标签过滤、详情弹窗（多张图片查看）
- **📝 Articles 文章页** — 文章列表，支持搜索、分类筛选、全文弹窗
- **👤 About 关于页** — 个人介绍、统计图表（渐变填充条）、时间线、语录、技能标签、联系方式
- **🖼 Gallery 相册页** — 图片网格，全屏查看器翻页浏览（键盘 ← → 切换），时间排序
- **🔒 编辑模式** — 支持添加/编辑/删除内容，拖拽排序，双击编辑个人信息
- **🌙 暗色模式** — 一键切换明暗主题
- **⌨️ 键盘快捷键** — `E` 编辑模式、`1`-`5` 切换页面、`Esc` 关闭弹窗
- **🖱 详情与灯箱** — 点击卡片查看详情，图片支持多张展示与点击放大
- **☁️ 云端备份** — 支持连接 GitHub Gist 自动同步数据
- **💾 本地备份** — 导出/导入 JSON 备份文件

## 🛠 技术栈

纯前端 SPA，无框架依赖：

- **HTML** — 语义化结构
- **CSS** — 自定义属性主题、Grid 布局、渐变装饰、毛玻璃效果、响应式设计
- **JavaScript** — MVC 模式、localStorage 持久化、IntersectionObserver 入场动画、拖拽排序、GitHub Gist API 同步

## 🖥 响应式布局

- **桌面端 (≥1024px)** — 三列作品网格、大号排版、宽间距、优化的 About 双栏布局
- **平板 (768-1024px)** — 两列网格、适中间距
- **手机 (<768px)** — 单列布局、紧凑导航、触摸优化

## 🚀 使用

所有数据存储在浏览器 localStorage 中。点击导航栏 `✎` 按钮进入编辑模式即可管理内容。

### 数据备份

1. 进入编辑模式后点击右下角 `💾` 备份按钮
2. **导出 JSON** — 下载备份文件到本地
3. **导入 JSON** — 从备份文件恢复数据
4. **GitHub Gist 同步** — 输入 Personal Access Token（需 `gist` 权限）实现自动云端同步
