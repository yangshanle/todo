# 个人作品网页优化计划

## 一、需求分析

用户提出以下优化需求：

### 1. 作品卡片布局优化
- 将现有整齐排列的卡片调整为不规则拼图式布局
- 实现不同尺寸的卡片（正方形、长方形等）无间隙地组合排列
- 允许较大卡片右侧同时放置2-3个较小卡片
- 形成视觉上富有层次感的非对称布局

### 2. 页面标题区域优化
- 在工作页、home页和article页的标题左右两侧添加装饰元素
- 填充当前空白区域，增强页面平衡感

### 3. 数据展示板块增强
- 改进"作品"、"文章"、"影像"和"技能"等数据的数字展示板块设计
- 添加视觉元素提升美观度和信息传达效果

### 4. 古诗板块调整
- 减小古诗板块的整体版面尺寸
- 增加经典代码片段、数学公式以及名言警句等多元化内容

### 5. 内容更新方式
- 新增和修改的内容采用固定形式呈现
- 无需实现动态更新功能

---

## 二、技术方案

### 1. 作品卡片拼图布局实现

**修改文件**: `src/css/washi.css`

采用 CSS Grid 实现不规则拼图布局：
- 使用 `grid-template-columns` 和 `grid-template-rows` 定义网格
- 通过 `grid-column` 和 `grid-row` 控制卡片跨越
- 添加不同尺寸的卡片样式类：`.card-large`, `.card-medium`, `.card-small`

### 2. 页面标题装饰元素

**修改文件**: `index.html`, `src/css/washi.css`

在标题区域添加装饰元素：
- 添加左右对称的装饰线条和几何图形
- 使用 SVG 或 CSS 绘制装饰图案
- 实现响应式适配

### 3. 数据展示板块设计

**修改文件**: `index.html`, `src/css/washi.css`, `src/js/app.js`

增强数据统计展示：
- 添加图标和动画效果
- 使用渐变背景和阴影增强视觉层次
- 添加数字递增动画

### 4. 古诗板块内容扩展

**修改文件**: `index.html`, `src/css/washi.css`, `src/js/app.js`

扩展内容类型：
- 添加代码片段展示区域
- 添加数学公式展示
- 添加名言警句
- 使用轮播切换不同内容类型

## 四、实现步骤

### 步骤 1：实现拼图式卡片布局
1. 修改 `.w-grid` 的 Grid 布局
2. 添加不同尺寸卡片的样式类
3. 修改 JavaScript 渲染逻辑支持不同尺寸

### 步骤 2：添加页面标题装饰
1. 在 HTML 中添加标题装饰容器
2. 创建装饰元素的 CSS 样式
3. 实现响应式适配

### 步骤 3：增强数据展示板块
1. 修改统计卡片的 HTML 结构
2. 添加视觉增强效果（渐变、阴影、图标）
3. 实现数字动画效果

### 步骤 4：扩展古诗板块内容
1. 修改诗歌轮播区域的结构
2. 添加代码片段、公式、名言等内容
3. 实现内容类型切换

---

## 五、风险与注意事项

1. **布局兼容性**: 拼图布局在小屏幕上需要回退到单列布局
2. **性能考虑**: 过多装饰元素可能影响页面加载速度，需优化
3. **代码复用**: 确保新增代码与现有架构保持一致

---

## 六、预期效果

完成后页面将具有以下特点：
- 不规则拼图式作品卡片布局，视觉层次丰富
- 标题区域左右对称装饰，页面平衡感增强
- 数据展示板块具有现代感的视觉效果
- 古诗板块内容多元化，包含代码、公式、名言等
