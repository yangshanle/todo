# Portfolio 使用说明

## 快捷键

| 按键 | 功能 |
|------|------|
| `E` | 切换编辑模式（需输入密码） |
| `1` | 首页 Home |
| `2` | 作品 Works |
| `3` | 文章 Articles |
| `4` | 关于 About |
| `5` | 相册 Gallery |
| `Esc` | 关闭当前弹窗 |
| `←` / `→` | Gallery 查看器中上一张/下一张 |

快捷键在输入框或弹窗打开时自动禁用，避免误触。

---

## 页面功能

### 🏠 Home 首页
- 展示头像、姓名、头衔、简介
- 精选作品（前 3 个）、最新文章（前 3 个）、技能标签
- 点击作品/文章直接跳转查看详情
- 装饰性几何图形与公式带视差滚动效果
- 语录气泡随机展示

### 💼 Works 作品页
- 作品以卡片网格展示（2 列）
- 顶部搜索框可搜索标题、描述、标签
- 标签过滤按钮筛选作品
- 非编辑模式：点击卡片打开详情弹窗
- 编辑模式：拖拽调整顺序、✎ 编辑、✕ 删除

### 📝 Articles 文章页
- 文章列表按日期排列
- 顶部搜索框可搜索标题、分类
- 分类过滤按钮
- 非编辑模式：点击文章打开全文弹窗
- 编辑模式：拖拽调整顺序、✎ 编辑、✕ 删除

### 👤 About 关于页
- 头像 + 个人介绍
- 统计卡片（作品/文章/影像/技能数量，渐变填充条）
- 当前状态（绿色指示灯 + 正在做的事）& 个人哲学签名
- 趣味数据（咖啡、提交数、项目数等可自定义）
- 时间线（工作经历 + 教育经历，按年份排序）
- 语录（随机展示，可管理多条）
- 技能标签、联系方式、社交链接

### 🖼 Gallery 相册页
- 图片以卡片网格展示（与作品页样式一致），按日期倒序排列
- 每张卡片可包含：图片、标题、日期、描述、标签、链接
- 非编辑模式：点击卡片或图片打开全屏查看器，带上下翻页浏览
- 全屏查看器支持键盘 `←` `→` 翻页，`Esc` 关闭
- 编辑模式：拖拽调整顺序、✎ 编辑、✕ 删除

## 编辑模式

1. 点击导航栏 `✎` 按钮
2. 输入密码（默认: `1`），点击确认
3. 进入编辑模式后：
   - 右下角出现 `+` 浮动按钮，可添加作品/文章/图片
   - 各页面出现编辑/删除按钮
   - 双击首页文字可修改个人信息
   - 点击技能/联系方式/社交链接可修改
   - 时间线、语录、状态、趣味数据进入可编辑状态
4. 再次点击 `🔓` 按钮退出编辑模式

---

## 数据存储

所有数据存储在浏览器的 **localStorage** 中，清除浏览器数据会导致内容丢失。建议定期备份。

---

## 提示

- 暗色模式：点击导航栏 `🌙` 按钮切换
- 相册初始为空，需进入编辑模式添加图片
- 图片支持 URL 和本地上传两种方式
